## 使用指南

### 对于学习者

1. **开始学习**
   ```bash
   cd learn/
   cat README.md  # 阅读总体介绍
   ```

2. **按顺序学习主题**
   ```bash
   cd 01-typescript-advanced/01-type-system/
   cat README.md  # 主题概述
   cat theory/guide.md  # 理论学习
   cat exercises/problems.md  # 练习题
   cat projects/requirements.md  # 实战项目
   ```

3. **跟踪进度**
   - 使用每个主题 README 中的学习检查清单
   - 在 `STATUS.md` 中查看整体进度

### 对于内容贡献者

1. **选择主题**
   ```bash
   cd learn/
   cat STATUS.md  # 查看哪些主题待完成
   ```

2. **填充内容**
   ```bash
   cd [category]/[topic]/
   # 按照 CONTRIBUTING.md 的指南填充内容
   ```

3. **使用主题 01 作为参考**
   ```bash
   # 查看已完成的主题 01
   cat 01-typescript-advanced/01-type-system/theory/guide.md
   ```
